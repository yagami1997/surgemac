# Neo Ruleset 2026 Migration URLs

This document is a migration-ready list of absolute rule URLs under `neorulset26/`.

Share these links directly with anyone migrating to the 2026 ruleset stack.

## AI Usage Note

This file is intended to be AI-friendly.

If you send this document to an AI assistant, the AI should be able to:

- read the absolute rule URLs directly
- map them into a Surge configuration
- generate `RULE-SET,...` lines from these links
- place the links into the correct policy groups if you also provide your strategy group names

Recommended instruction to send with this file:

```text
Use the absolute rule URLs in this document to build or update my Surge configuration.
Keep the URLs exactly as written.
Generate valid RULE-SET entries directly from these links.
If a policy name is not explicitly provided, ask me before assigning one.
```

If your friends already know their policy names, they can also tell the AI something like:

```text
Use these rule URLs and write them into my Surge config.
Map each RULE-SET to my existing policy groups.
Do not rewrite or shorten the URLs.
```
