# Modules

All shared Surge modules belong in this directory.

## Rules

- Place every tracked `.sgmodule` file here.
- Do not add `.sgmodule` files to the repository root.
- Keep module filenames stable once they are shared externally.
- Give each new module a clear, product-like name.
